# AI Workflows Hub

A centralized repository containing multiple **AI-powered automation workflows and agent-based systems** designed to reduce repetitive business tasks and improve productivity.

This hub serves as a growing collection of practical AI applications built across domains such as HR automation, developer productivity, and data processing.

---

## Purpose

The goal of this repository is to:

- Showcase real-world AI automation projects
- Demonstrate end-to-end AI system design
- Build scalable and reusable AI workflows
- Experiment with NLP, ML, and agent-based automation

Each folder represents an independent AI project with its own documentation and setup instructions.

---

## Projects

### 1. AI Resume Screening & Candidate Ranking System

An AI-powered system that:

- Parses resumes (PDF/DOCX)
- Extracts structured candidate information
- Matches candidates against job descriptions using semantic similarity
- Scores and ranks applicants
- Stores data using MongoDB
- Exports shortlisted candidates

**Folder:** `ai-resume-screening-system`
